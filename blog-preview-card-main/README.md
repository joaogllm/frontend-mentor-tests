## Overview

This challenge helped me practice building a self-contained card component using semantic HTML. A few things I paid attention to:

- Using <article> for the card wrapper instead of a generic <div>, since it represents a self-contained piece of content
- Loading a variable font locally via @font-face with the correct weight range syntax (300 900)
- Using box-shadow with no blur to achieve the hard-shadow design style common in Frontend Mentor challenges
- Adding padding to the body to prevent the card from touching the screen edges on mobile
