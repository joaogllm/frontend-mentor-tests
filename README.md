# Frontend Mentor Projects 🚀

This repository contains a collection of projects I built while studying and practicing front-end development through **Frontend Mentor** challenges.

The goal of these projects is to improve my skills in **HTML, CSS, JavaScript**, and modern front-end practices by building real-world UI components and layouts.

---

## 🧠 What is Frontend Mentor?

[Frontend Mentor](https://www.frontendmentor.io/) provides real-world front-end challenges that help developers improve their coding skills by building projects based on professional designs.

---

## 🛠️ Technologies Used

- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, responsive layouts)
- JavaScript (ES6+)
- Git & GitHub
- Mobile-first workflow

Some projects may also include:

- CSS animations
- Accessibility best practices
- Responsive design techniques
