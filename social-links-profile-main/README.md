## Overview

- A simple but polished social links profile card built as a Frontend Mentor challenge. The goal was to replicate the provided design faithfully — centering a profile card with an avatar, name, location, bio, and a list of social links — while applying good practices like semantic HTML, CSS custom properties, BEM naming, and proper hover/focus states.

### AI Collaboration

I used **Claude (Anthropic)** as an AI assistant throughout this project.

**How I used it:**

- Generated a modern CSS reset as a starting point
- Requested a code review after finishing my implementation — Claude identified issues like a duplicate `body` block

**What worked well:**
Having a second pair of eyes on the code review was the most valuable part. It caught subtle but important issues — like the duplicate `body` rule.

**What didn't work as well:**
AI-generated code is a good starting point, but it still requires careful reading and adaptation to fit your specific project context. Blindly copying suggestions without understanding them would defeat the purpose of the challenge.
